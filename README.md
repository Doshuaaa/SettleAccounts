# 정산하자! (부제: SettleAccounts)

## 프로젝트 개요 
- 모임 이후 정산하기 귀찮은 당신을 위한 간단 정산 앱

## 개발 기간
- 2024.02.23 ~ 2024.03.11

## 앱 내 기능
  - 모임 참여 인원 수와 이름 입력
  - 모임 활동 수와 활동명 입력
  - 각 활동 별 비용 입력 및 참여 인원 선택 
  - 은행명과 계좌 번호 입력
  - 결과 확인 후 클립보드에 텍스트 복사

## 사용된 기술
  - MVVM
  - LiveData
  - DataBinding
  - ViewModel
  - lifecycle
